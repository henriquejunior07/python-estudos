# Aula 06 - Tipos primitivos e strings

## O que aprendi
- input() sempre retorna string
- string é texto dentro de aspas
- int(), float(), bool() são conversões de tipo
- métodos de string retornam True ou False

## Métodos estudados
- isalpha() → só letras
- isnumeric() → só números
- isalnum() → letras e números
- isupper() → maiúsculas
- islower() → minúsculas

## Dificuldade
- entender diferença entre erro e False
